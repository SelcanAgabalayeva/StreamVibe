# 🎬 StreamVibe

StreamVibe is a Spring Boot web application for browsing and managing movies and TV shows.

---

## 🚀 Features
- 🎥 Movies & TV shows management
- 📂 Category system
- 🔍 Search functionality
- 📄 Detail pages
- ⚙️ RESTful architecture
- 🗄️ MySQL database integration

---

## 🛠️ Tech Stack
- Java 17
- Spring Boot
- Spring MVC
- Spring Data JPA
- Hibernate
- Maven
- MySQL
- Thymeleaf (optional)

---

## 📁 Structure
- controller/
- service/
- repository/

- dto/

- config/
