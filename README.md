# 🎬 StreamVibe

StreamVibe is a Spring Boot web application for browsing and managing movies and TV shows.

---

## 🚀 Features

- 🎥 Movies & TV shows management
- 📂 Category system
- 🔍 Search functionality
- 📄 Detail pages
- ⚙️ RESTful architecture
- 🗄️ MySQL database integration

---

## 🛠️ Tech Stack

- Java 17
- Spring Boot
- Spring MVC
- Spring Data JPA
- Hibernate
- Maven
- MySQL
- Thymeleaf (optional)

---

## 📁 Project Structure


controller/
service/
repository/
model/
dto/
exception/
config/


# Enter project folder
cd streamvibe

# Run with Maven
mvn spring-boot:run
⚙️ Requirements
Java 17+
Maven 3+
PostgreSql running locally or remote
